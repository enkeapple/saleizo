# Learning Kit

User-invoked skills for learning something fast and proving you actually got it — translate confusion into plain language, expose the gaps you can't see, and build a plan that reaches a concrete outcome.

## Quickstart

- Not sure where to start? Invoke `router` — name your learning moment and it points you at the one skill to use.
- Stuck on confusing material → `confusion-translator`. Think you've mastered it → `hidden-gaps-detector`. Need a schedule → `plan-architect`.
- Install it first — see [Installation](#installation).

## How it works

Each skill is invoked by name and drives an interactive session — it asks you questions, makes you explain or apply the concept, and gates on real comprehension rather than handing you a summary to nod along to.

## Installation

### Claude Code

```text
/plugin marketplace add enkeapple/obilyk-skills
/plugin install learning-kit@obilyk-skills
```

## What's Inside

A family of complementary learning moves — translate jargon, destroy the learning curve, force a Feynman explanation, detect hidden gaps, simulate real mistakes, and architect a day-by-day plan — with `router` as the index that picks the right one.

## Philosophy

Reading an explanation feels like learning but rarely is. These skills make you produce — explain, apply, fail, retry — because that is what actually moves understanding, and they gate on it instead of trusting "makes sense".
