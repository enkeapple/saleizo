# Craft Kit

Craft skills for the parts of engineering that aren't a pipeline step: designing a module worth depending on, reviewing an architecture for where depth is missing, and making prose read like a human wrote it.

## Quickstart

- Designing or reshaping a module? Start with `codebase-design` for the shared deep-module vocabulary, then `improve-codebase-architecture` for a review pass.
- Cleaning up writing? `tightening-prose` for technical docs, `humanizing-prose` for an article or post.
- Install it first — see [Installation](#installation).

## How it works

The skills are user-invoked: name the task ("design this module", "de-slop this draft") and Claude loads the matching skill. Each one carries a fixed catalog or vocabulary so the same pass runs the same way every time.

## Installation

### Claude Code

```text
/plugin marketplace add enkeapple/obilyk-skills
/plugin install craft-kit@obilyk-skills
```

## What's Inside

- **design** — a shared vocabulary for deep modules and an architecture-review pass that surfaces where interfaces are too shallow.
- **prose** — two de-slopping passes: a terse technical one and a narrative/publication one, with deliberately opposite carve-outs.

## Philosophy

Depth and clarity don't happen by default — they need a fixed pass the model's own taste keeps skipping. These skills encode that pass so it runs the same way every time, instead of relying on the agent to remember.
