# Phrase Catalog — register-agnostic AI tells to cut

Shared by `tightening-prose` (technical-doc register) and `humanizing-prose` (narrative / article register). These phrase-level tells are slop in **both** registers — the only thing that differs is the empty-adverb carve-out (technical docs keep meaning-bearing adverbs and informative hedges; narrative prose does not). Each skill's own `phrases.md` states its register note; this file is the single source of truth for *what* to cut. The catalog is the value: a model removes the obvious tells from taste but misses the long tail without a list to check against.

## Throat-clearing openers

Announcement phrases before the point. Cut the announcement, state the point.

- "Here's the thing:" / any "Here's what / this / that / why …" construction
- "It turns out" / "The truth is," / "The real X is"
- "The reality is" / "The uncomfortable truth is"
- "Let me be clear" / "I'm going to be honest" / "Can we talk about"
- "When it comes to …" / "In today's world / fast-paced world"

## Emphasis crutches & performative

Add no meaning. Delete.

- "Full stop." / "Period." / "Let that sink in."
- "Make no mistake" / "This matters because" / "Here's why that matters"
- "I promise" / "They exist, I promise" / "creeps in" (manufactured intimacy)
- "Plot twist:" / "Spoiler:" / "Hint:" / "You already know this, but"

## Empty adverbs & softeners

The words below add no information. How hard to apply depends on register — see the calling skill's register note (a technical doc keeps meaning-bearing adverbs and informative hedges; narrative prose cuts freely).

- "really", "just", "literally", "genuinely", "honestly", "simply", "actually"
- "deeply", "truly", "fundamentally", "inherently", "inevitably"
- "interestingly", "importantly", "crucially"

Filler phrases in the same class:

- "At its core" / "When it comes to" / "At the end of the day"
- "It's worth noting" / "In today's X" / "In a world where"

## Meta-commentary

Self-referential asides that announce structure instead of advancing it. Delete.

- "As we'll see" / "Let me walk you through" / "In this section, we'll"
- "I want to explore" / "The rest of this explains"

## Telling instead of showing

Announcing significance or difficulty rather than demonstrating it.

- "This is genuinely hard" / "This is what X actually looks like" / "actually matters"

## Vague declaratives

Claim importance without naming the specific thing. Replace with the specific thing, or cut.

- "The stakes are high" / "The consequences are real"
- "The implications are significant" / "The reasons are structural" / "This is the deepest problem"
