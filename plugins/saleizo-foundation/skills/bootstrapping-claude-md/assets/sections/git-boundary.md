## Git boundary

The human owns the commit. Autonomous: Read/Edit/Write in the working tree, read-only git, lint/typecheck/tests. Never without explicit instruction this turn: `git commit`/`push`/`reset --hard`/branch ops/secret edits. On a fully-complete, verified change, propose a one-line Conventional Commit; the human runs it.
